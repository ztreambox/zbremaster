Welcome to ztreambox 12.04.01

ztreambox is a collection of software applications that allows the easy operation of a Media Center PC running the Linux operating system.


ztreambox is in large parts already pre-configured and ready for use directly after installation, so that especially the newbie does not need to perform any configuration tasks.


ztreambox uses the popular XBMC as a Media Center application.
